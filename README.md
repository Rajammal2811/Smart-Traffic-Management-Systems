# Smart-Traffic-Management-Systems 

## Project Overview
The Smart Traffic Management System is designed to reduce traffic congestion by managing traffic signals based on vehicle density. It improves traffic flow and gives priority to emergency vehicles.

## Features
- Smart Traffic Signal Control
- Vehicle Density Monitoring
- Emergency Vehicle Priority
- Traffic Reports
- Admin Dashboard

## Technologies Used
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript

## Future Enhancements
- AI-based Vehicle Detection
- Number Plate Recognition
- Live Traffic Monitoring
